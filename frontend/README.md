# FotoEduc Frontend

Frontend scaffold com Next.js (App Router), Tailwind e páginas de autenticação.

Como rodar localmente:

1. Copie as variáveis (se necessário):

- NEXT_PUBLIC_API_URL=http://localhost:3001

2. Instale dependências:

cd frontend
npm install

3. Rodar em dev:

npm run dev

OBS: Frontend espera a API backend rodando em http://localhost:3001 conforme padrão do scaffold.
